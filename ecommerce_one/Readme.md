# ============ ReactJS ==========
* Init React Project
  > node -v (ok/not(install node))
  > npx create-react-app project_name // need internet
  > cd project_name // goto root project
  > npm start // run project 
* delete node_modules
  > npm install // re get node_modules in project
* React route
  > npm install react-router-dom
* React with API 
  > npm install axios
